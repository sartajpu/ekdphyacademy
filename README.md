# 📚 EKD Physics Academy Website

यह एक पूरी तरह से फंक्शनल वेबसाइट है जिसे छात्रों के लिए बनाया गया है। इसमें फ्रंटएंड के साथ-साथ बैकएंड डेटाबेस और क्लाउड होस्टिंग का बेहतरीन उपयोग किया गया है।

🔗 **लाइव डेमो लिंक:** [यहाँ क्लिक करें]([(**https://ekdphyacademy.web.app/**)) (अपनी असली लिंक डालें)

## 🛠️ टेक स्टैक (Tech Stack)
- **Frontend:** HTML5, CSS3, JavaScript (Mobile Responsive)
- **Hosting:** Firebase Hosting
- **Database:** Supabase (PostgreSQL)

## ⚙️ मैंने इस प्रोजेक्ट में क्या-क्या किया है?
1. **Firebase Hosting:** वेबसाइट की स्पीड और परफॉरमेंस को बढ़ाने के लिए इसे फायरबेस पर होस्ट किया है। इसकी कॉन्फ़िगरेशन फाइलें `firebase.json` और `.firebaserc` प्रोजेक्ट में शामिल हैं।
2. **Supabase Integration:** वेबसाइट के डेटा (जैसे छात्र का डेटा, इन्क्वायरी फॉर्म, या कोर्स डिटेल्स) को मैनेज करने के लिए **Supabase (PostgreSQL)** क्लाउड डेटाबेस का इस्तेमाल किया गया है।
3. **Responsive Design:** `mobile.css` का उपयोग करके वेबसाइट को मोबाइल और कंप्यूटर दोनों के लिए पूरी तरह अनुकूलित (Responsive) बनाया गया है।

## 💻 लोकल सेटअप (Local Setup)
यदि आप इसे अपने कंप्यूटर पर चलाना चाहते हैं:
1. रिपोजिटरी क्लोन करें: `git clone https://github.com`
2. `index.html` को किसी भी ब्राउज़र में या Live Server एक्सटेंशन के साथ खोलें।
